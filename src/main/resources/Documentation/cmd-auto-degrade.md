@PLUGIN@ auto-degrade
=====================

NAME
----
@PLUGIN@ auto-degrade - Disable DEGRADED mode

SYNOPSIS
--------
```
ssh -p @SSH_PORT@ @SSH_HOST@ @PLUGIN@ auto-degrade
```

DESCRIPTION
-----------
Disables DEGRADED mode so caches function normally.

ACCESS
------
Caller must be a member of the privileged 'Administrators' group.
